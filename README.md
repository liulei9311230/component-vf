# component-vf  组件库
公司自主研发框架vf.js（基于vuejs和angularjs的MVP模式）开发的一套组件库


# 组件化开发的优点
> 1.块状化结构，减少css 的书写，并且方便扩展

> 2.标记鲜明，容易维护
